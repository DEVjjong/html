select / option
		- 항목리스트 에서 한개 또는 여러개를 선택
		- 여러개를 선택시 select 태그에서 multiple="multiple" 속성을 설정
		- 화면에 표시할 항목의 갯수 : select태그에서 size="6" 속성을 설정
		- 초기 실행시 표시될 항목 : option태그에서 selected="selected" 속성을 설정
		- option태그의 value속성은 서버로 전송되는 값을 설정
		
		input태그의 value속성은 초기실행시 화면에 표시되는 문자
		input태그는 한줄로만 입력이 가능
		여러줄을 입력하기 위해서는 textarrea태그를 사용
		가로길이 : cols = ""50"" 속성
		세로줄이 : rows -"10" 속성
